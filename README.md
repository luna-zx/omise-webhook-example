# ตัวอย่างการรับ charge events จาก Omise Webhook

> Dependencies :
- Elysia.js
- Omise.js

### วิธีการทดสอบ

```
git clone https://github.com/luna-zx/omise-webhook-example.git
cd omise-webhook-example
bun install
npm run dev
```

อย่าลืมแก้ `OMISE_PUBLIC_KEY` และ `OMISE_SECRET_KEY` ในไฟล์ `.env` ด้วยน้า  

```
OMISE_PUBLIC_KEY = 
OMISE_SECRET_KEY = 
```
